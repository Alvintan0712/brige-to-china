### 1.代码命名规范

- 变量规范:
```// 变量统一使用let声明, 使用小驼峰语法, 如作用域内变量初始值无变化切换为常量定义, 例:
let current;
let phoneInputRef = useRef<Element>(null);
```

- 常量规范: 
```
// 静态常量统一使用大写命名,单词使用下划线拼接,密集常量定义使用单Const 定义,逗号分割,枚举同常量,例: 
// 静态常量
const USER_INFO = *;
// 动态常量
const url = '/api/userinfo'
// 常量集
const userId = 10001,
      userName = 'tester',
      Phone = 13800138000;
```
